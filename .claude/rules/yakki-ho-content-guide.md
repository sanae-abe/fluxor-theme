# 薬機法対応 コンテンツ入力ガイドライン

Shopify 管理画面（メタフィールド・テーマエディタ）でコンテンツを入力する際の薬機法遵守ルール。

---

## 対象フィールドとリスクレベル

| フィールド | 画面 | リスク |
|---|---|---|
| `key_ingredient.description` | 商品ページ 成分ツールチップ | HIGH |
| `roles[].description` | 成分記事 役割欄 | HIGH |
| `product.metafields.custom.product_benefits` | 商品ページ 効能リスト | HIGH |
| `usage_tips[].description` | 成分記事 使い方説明 | MEDIUM |
| `compatible_ingredients[].reason` | 成分記事 相性説明 | MEDIUM |
| `pregnancy_note` | 成分記事 妊娠注意 | MEDIUM |
| テーマエディタ `disclaimer`（用語定義） | サイト全体フッター注記 | MEDIUM |

---

## 絶対禁止表現

以下の表現が含まれるコンテンツは **公開しないこと**。

### 医薬品的な機序・作用の説明
- 「メラニンを抑制する／分解する」
- 「ターンオーバーを促進する」
- 「細胞を活性化する」
- 「炎症を抑える」「抗炎症」
- 「殺菌」「抗菌」「消毒」
- 「コラーゲンを増やす／生成を促進する」
- 「ヒアルロン酸が真皮に浸透して」
- 「育毛」「発毛」「脱毛を防ぐ」

### 疾病・症状への言及
- 「アトピー」「湿疹」「皮膚炎」「かぶれ」
- 「ニキビを治す」「ニキビに効く」「ニキビ菌を殺す」
- 「シミが消える」「シワが消える」
- 「傷を治す」「かゆみを止める」

### 誇大表現・根拠のない比較
- 「〇〇比 ◯% アップ」（独自比較）
- 「No.1」「最強」「最高峰」（根拠なし）
- 「医師推薦」「皮膚科医が選ぶ」（証拠なき推薦）
- 「◯週間で効果が出る」

---

## OK な表現（化粧品効能の範囲内）

| 禁止（医薬品的） | OK（化粧品の範囲） |
|---|---|
| メラニンを抑制する | うるおいを与え乾燥によるくすみを防ぐ |
| コラーゲンを増やす | ハリ感を与える／ハリ感をサポートする |
| シミが消える | 肌にうるおいを与え、明るい印象を保つ |
| シワが消える | 目立ちにくくする |
| ターンオーバーを促進 | キメを整える |
| ニキビを治す | 洗浄により肌を清浄にし、ニキビを防ぐ（洗浄系のみ）|
| 細胞を活性化 | 肌本来のうるおいを保つ |
| 浸透して真皮へ | 角質層まで浸透する |

## 成分説明の書き方ルール（`roles[].description`・`key_ingredient.description`）

### NG パターン
```
ナイアシンアミドは、メラノサイトからケラチノサイトへのメラニン転送を阻害し、
シミ・くすみを改善します。また、皮脂分泌を抑制し、毛穴を引き締める効果があります。
```

### OK パターン
```
肌のキメを整えるのに役立つ成分として、多くのスキンケア製品に配合されています。
うるおいを与えながら肌の状態を整えます。
※本記載は整肌・保湿成分としての役割の説明です。
```

**ポイント**:
- 薬理的な機序の説明は書かない
- 「整肌」「保湿」「うるおい」「キメを整える」「肌状態を整える」の言葉を基軸にする
- 成分の「働き」ではなく「配合目的」として説明する

---

## `pregnancy_note`（妊娠・授乳中の方へ）の書き方

### NG
```
レチノールは胎児に影響を与える可能性があり、妊娠中の使用は禁忌です。
```

### OK
```
妊娠中・授乳中の方は、ご使用前に産婦人科医または皮膚科医にご相談ください。
```

- 「禁忌」「危険」「影響を与える」等の医学的断言は避ける
- 必ず「専門医にご相談ください」で締める

---

## テーマエディタ `disclaimer`（用語定義）フィールドの編集方針

テーマエディタ > テーマ設定 > 用語定義 に入力する文言は以下の基準で管理する。

**現在の承認済み定義（変更時は法務確認が必要）**:
- 浸透：角質層まで。
- エイジングケア：年齢に応じた潤いによるお手入れのこと。
- ニキビケア：洗浄により肌を清浄にし、ニキビを防ぐこと。
- 毛穴ケア：潤いを与え、キメを整えることで毛穴を目立たなくすること。
- 美白、透明感：うるおいを与え乾燥によるくすみを防ぐこと、またはキメが整うことによる肌印象のこと。
- 配合成分（ペプチド、アゼライン酸等）の表記は、すべて整肌・保湿成分としての目的。

## 成分記事 公開前チェックリスト

成分記事を公開する前に以下を確認すること。

- [ ] `roles[].description` に「メラニン」「ターンオーバー」「炎症」「コラーゲン増」の語がない
- [ ] `key_ingredient.description` に薬理的機序の説明がない
- [ ] `pregnancy_note` が「専門医にご相談ください」で終わっている（断言していない）
- [ ] `compatible_ingredients[].reason` に「吸収を高める」「浸透を助ける（真皮）」等の表現がない
- [ ] 成分記事の免責ブロック（ `disclaimer` ブロック）が配置されている
- [ ] 記事本文（`article_content`）に「〇週間で変化」「効果が出る」等の表現がない

## 参考：化粧品の効能として認められている56効能（抜粋）

日本の化粧品は薬機法により、認められた効能以外を謳うことができない。

主な認可効能:
- 頭皮、毛髪を清潔にする
- 香りにより毛髪・頭皮の不快臭を抑える
- 頭皮・毛髪をすこやかに保つ
- 毛髪にはり、こしを与える
- 頭皮、毛髪にうるおいを与える
- 皮膚をすこやかに保つ
- 皮膚にうるおいを与える
- 皮膚を柔らかくする
- 皮膚にはりを与える
- 皮膚のキメを整える
- 皮膚を清潔にする
- ニキビを防ぐ（洗浄）
- **「シミ・そばかすを防ぐ」は医薬部外品のみ**

全56効能は消費者庁「薬用化粧品の効能の範囲」を参照。
